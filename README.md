# 🎬 BookYourShow - Movie Ticket Booking Application

BookYourShow is a full-stack MERN (MongoDB, Express.js, React.js, Node.js) web application that allows users to browse movies and book tickets online.

---

## 🚀 Features

* 🔐 User Authentication (Login & Register)
* 🎥 Browse Movies
* 🎟 Book Movie Tickets
* 🪑 Seat Selection (future scope)
* 📜 Booking History (future enhancement)
* ⚙️ Admin Dashboard (future)

---

## 🛠 Tech Stack

### Frontend

* React.js
* HTML5, CSS3, JavaScript

### Backend

* Node.js
* Express.js

### Database

* MongoDB

## ⚙️ Installation & Setup

### 1. Clone Repository

git clone https://github.com/sushilrao39/BookYourShow.git

### 2. Install Dependencies

Client:
cd client
npm install

Server:
cd server
npm install

---

### 3. Run Application

Start Backend:
cd server
npm start

Start Frontend:
cd client
npm start
